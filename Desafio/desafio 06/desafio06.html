Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termor para descobrir qual tag decera utilizar em cada caso

  • Parágrafos
    • Use a tag <p>
  • Títulos
    • Use a tag <h1>
  • Imagens
    • Use a tag <img>
  • Abreviações
    • Use a tag <abbr>
  • Importância
    • Use a tag <strong>
  • Ênfase
    • Use a tag <em>
  • Citação
    • Use a tag <q>
  • Quebra de linha
    • Use a tag <br>
